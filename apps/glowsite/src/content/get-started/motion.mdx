# How do I get a drive moving?

> Assumes you have the single axis EtherCAT starter kit hardware

## Step 1 - Download the generic front-end application to your PC
````shell
npm i
````
## Step 2 - Download and run GBC on the Raspberry Pi
Connect to the pi with ssh and password `raspberry`
````shell
ssh root@rpi-gb
````
Download GBC from github?
````shell
mkdir glowbuzzer_toolkit
````

## Step 3 - Download GBEM configure it for the test machine hardware
Download from github
````shell
git clone https://gitgub.com/glowbuzzer/GBEM

````
Open in your IDE and edit the following files:

machine config



````shell
````

## Step 4 - Run GBEM and make sure it connected to both GBC and the test hardware
````shell
./GBEM -c -ieth0
````
